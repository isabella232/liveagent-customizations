How can we help you?

Welcome to our help desk! If you’ve landed on this page, it means you have a question that’s above and beyond what is covered in our documentation and knowledgebase. Don’t worry … we’re here to help!

If you are an existing customer, please login before seeking support. If you are unsure which credentials to use or have any difficulties logging in, please read this guide for further advice.

We also ask that you read through our Help Desk Guidelines before posting so you know what to expect from our team. Additionally, we maintain a list of Known Issues so you can check and see if we are already aware of a particular problem before posting for help. Thank you for your cooperation!

Considering purchasing (but don't have a license key just yet)? That's awesome — we'd be happy to answer any questions you have about our products! If you don't already have a login you can send us a question quickly and easily via the pre-sales form found below.

  • Please note that we are unable to offer technical support via our pre-sales form. If you do not have a license key and are using one of our free plugins, you should post in our community forums instead.
  • Existing customers who already have a license key but have pre-sales questions about additional products can simply login and post a ticket as normal.
